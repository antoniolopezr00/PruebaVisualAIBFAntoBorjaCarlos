Random Forest

Es un algoritmo de aprendizaje automático que se basa en un conjunto de árboles de decisión. Durante el entrenamiento, selecciona aleatoriamente características y muestras de datos para construir cada árbol. Luego, utiliza votación o promedio para clasificación o regresión respectivamente. Es robusto, preciso y útil para conjuntos de datos grandes y complejos, aunque puede ser menos interpretable que un solo árbol de decisión.
